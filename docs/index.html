How it works

The game stores event rewards and progress in an accountDataDB.package file. By modifying that file you can trick the game into thinking you've unlocked all rewards.

  1. When an event is live, it lets you claim all rewards without doing a single quest. This works in online mode only, since to claim the rewards you need to be online. You can claim all rewards, even from weeks that are still locked, so you claim the rewards at your own risk. It's best to wait until all weeks are unlocked.
  2. If the event has already ended, you can still unlock the rewards, but it will work in offline mode only. Every time you start the game online it will replace the rewards data. It's more convenient to use mods instead.

Unlock your rewards

Select your UserSetting.ini file from Documents > Electronic Arts > The Sims 4*. Your browser will download a patched accountDataDB.package — place it in the same folder.

FAQ

Where is my UserSetting.ini?
Open File Explorer and go to Documents > Electronic Arts > The Sims 4. The file UserSetting.ini should be right there. If you don't see it, make sure you've run the game at least once. In some languages the folder is named differently (e.g. Die Sims 4, Les Sims 4).
Why do rewards disappear when I go online?
When you start the game online, EA's servers sync your actual event data and overwrite the local accountDataDB.package. For ended events, the rewards only persist in offline mode. For live events, the rewards are tied to your EA account permanently once claimed online.
Where do I put the downloaded file?
Place accountDataDB.package in the same folder as your UserSetting.ini — that's Documents > Electronic Arts > The Sims 4. If the file already exists, it will be replaced.
Is this safe? Can I get banned?
This tool modifies a local data file, not the game executable. There are no known cases of bans from using this type of tool. However, use it at your own risk. Your file never leaves your browser — all processing happens locally.
It says "too many accounts" — what do I do?
The template supports up to 10 accounts. If you have more, delete or rename your UserSetting.ini file, run the game once to generate a fresh one, then try again. The new file will only contain your current account.
